# Taxlator

Taxlator is a tax calculation platform with a frontend UI and a backend API.

## Structure
- `taxlatorUI/` - Frontend application (React/Vite/Typescript)
- `taxlatorAPI/` - Backend API (Node/Express/Database-MongoDB)

## Setup
1. Clone the repo: `git clone ...`
2. Install frontend dependencies: `cd taxlatorUI && npm install`
3. Install backend dependencies: `cd taxlatorAPI && npm install`
